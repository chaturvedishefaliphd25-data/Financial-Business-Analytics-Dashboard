# Financial-Business-Analytics-Dashboard
# 📊 Financial Business Analytics Dashboard

## Business Analytics Capstone Project

### Project Overview

This project focuses on analyzing financial data from Western countries using SQL, Microsoft Excel, Power BI, and Tableau. The objective was to clean, analyze, visualize, and generate business insights to support data-driven decision-making.

---

## Tools & Technologies

- SQL Server
- Microsoft Excel
- Power BI
- Tableau
- Agile Methodology

---

## Project Workflow

- Data Cleaning and Preparation
- Statistical Analysis
- SQL Database Creation
- Data Import into Power BI
- Dashboard Development
- Business Insight Generation

---

## Dashboard KPIs

- Product-wise Sales
- Segment-wise Profit
- Country-wise Sales
- Yearly Sales
- Quarterly Profit
- Top 2 Countries
- Bottom 3 Products
- Product Discount Analysis
- Year-over-Year Units Sold

---

## Business Insights

- Identified top-performing countries based on sales.
- Compared yearly sales and profit trends.
- Evaluated segment-wise profitability.
- Built interactive dashboards for executive reporting.
